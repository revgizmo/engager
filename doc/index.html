zoomstudentengagement Package Vignettes

Welcome to the vignettes for the zoomstudentengagement R package. These vignettes provide detailed workflows and examples for analyzing student engagement from Zoom transcripts.

  • Getting Started
    Basic setup and workflow overview for new users
  • Transcript Processing
    Load, process, and analyze Zoom transcript files
  • Roster Management
    Manage student rosters and clean name mismatches
  • Plotting and Analysis
    Create visualizations and analyze engagement patterns
  • Session Mapping
    Handle complex scenarios with multiple courses/sections
